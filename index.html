Welcome to Bootstrap-Spinner

Bootstrap-spinner gives you a customizable bootstrap spinner with support for cross-browser supports,bootstrap button UI,mouse scrolling, and many other highly used options.

Downloading the code locally

In some situations, you can't use Bootstrap-spinner from a CDN and you must include the files through your own static file servers.

  1. Download the code from GitHub and copy the dist directory to your project.

  2. Include the following lines of code in the

    section of your HTML.

    <link href="path/css/bootstrap-spinner.min.css" rel="stylesheet" />

    <script src="path/js/bootstrap-spinner.min.js"></.script>

  3. Initialize Bootstrap Spinner on the <input type="text"> element that you want to make awesome. <script type="text/javascript">

    $(document).ready(function () { $('#txtrange').range({ InitValue: 200, minValue: 150, maxValue: 350, allowDecimal: true, decimalPrecision: 3, step: 2.2 }); });

    </script>

  4. Check out the examples page to start using the additional features of Bootstrap Spinner.
